# Stack and Queue

Can be implemented with:
- array; or
- linked list; or
- Standard Template Library (STL)

## Stack

- LIFO
- last in first out
- push & pop
- top & bottom
- Usage: compiler - managing curly braces etc

## Queue

- FIFO
- first in first out
- front & back
- popFront
- Usage: waiting lines
